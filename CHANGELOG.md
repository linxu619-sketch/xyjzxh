# 更新日志 CHANGELOG

本文件记录信阳市建筑装饰装修协会网站的**每一次修改**。
版本号遵循 [语义化版本 SemVer](https://semver.org/lang/zh-CN/)：`主版本.次版本.补丁`
- **补丁 +1**（0.1.0 → 0.1.1）：修 bug、文案/样式微调，不改功能
- **次版本 +1**（0.1.1 → 0.2.0）：新增功能、新页面
- **主版本 +1**（0.x → 1.0.0）：重大改版或不向后兼容的改动

版本号始终与 `package.json` 的 `version` 字段保持一致。

格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/)。

发布节奏：每次有意义的改动**直接新建一个版本号条目并提交**，不使用「未发布」缓冲区。

## [0.3.2] - 2026-05-30

### 修复（关键）
- **修复 `启动.bat` 双击启动失败**：根因是该 `.bat` 被存为 UTF-8，中文版 Windows 的 `cmd.exe` 用 GBK 解析时多字节中文导致命令被切碎、报「不是内部或外部命令」、服务器无法启动。改为 **GBK(cp936) 编码 + CRLF**，并去掉无效的 `chcp 65001`。实测双击场景下服务器第 2 秒即就绪。
- 新增 `.gitattributes`：`*.bat -text`，防止 git 改动 `.bat` 字节。
- 在 `CONVENTIONS.md` 记录「`.bat` 必须存 GBK」约定及改后重转命令。
